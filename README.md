# Java Memory Game

A console-based memory matching game built using Java.

## Features
- Randomly shuffled cards
- Input validation
- Match detection
- Game win condition

## Technologies Used
- Java
- Arrays
- ArrayList
- Collections

## How to Run
1. Compile: javac memgame.java
2. Run: java memgame
